# Claude Code Skills

Personal skill collection for Claude Code.

## Skills

- **bem-class-names-only** — Enforce BEM semantic class naming, forbid Tailwind utilities
- **nextjs-hydration-rules** — Hydration safety rules for Next.js App Router

## Usage

Clone into your project or global skills directory:

```bash
# Project-level
git clone https://github.com/haozhang46/claude-skills.git .claude/skills

# Global
git clone https://github.com/haozhang46/claude-skills.git ~/.claude/skills
```

Or pick individual skills:

```bash
cp -r claude-skills/bem-class-names-only your-project/.claude/skills/
```
