# Claude Code Skills

Personal skill collection for Claude Code.

## Skills

**Custom:**
- **bem-class-names-only** — BEM semantic naming, Tailwind `@apply` in CSS, no utilities in HTML
- **nextjs-hydration-rules** — 7 hydration safety rules for Next.js App Router
- **dto-mapper-layer** — Enforce mapper/DTO transform before storing API data in state

**Third-party (via install.sh):**
- **vercel-react-best-practices** — React/Next.js performance optimization from Vercel Engineering (40+ rules)

## Usage

```bash
git clone https://github.com/haozhang46/claude-skills.git
cd claude-skills
./install.sh  # fetch third-party skills
cp -r . your-project/.claude/skills/
```
